
This directory contains all of the files necessary to build this example on
the Linux operating system using gcc (3.0+ is supported).

compile - A script for compiling a statically VM-linked version of this example
link - A script for linking a statically VM-linked version of this example

compile-dyn - A script for compiling a dynamically VM-linked version of this example
link-dyn - A script for linking a dynamically VM-linked version of this example

When running the dynamically linked version, specify the entire path to the
virtual machine. For example, $JAVA_HOME/jre/lib/i386/server/libjvm.so
