
This directory contains all of the files necessary to build this JNI library for 
the Linux operating system using gcc (3.0+ is supported).

compile - A script for compiling a shared version of the library
link - A script for linking a shared version of the library

NOTE: You will have to set $JVM_LIB_DIR to the directory containing libjvm.so, such as: $JAVA_HOME/jre/lib/i386/client
